# 🌐 Abhishek Singh – Developer Portfolio

A modern, responsive portfolio website built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, and **GSAP** to showcase my skills, projects, education, and professional experience. The website features smooth animations, responsive design, and a built-in contact form powered by **Resend**.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/singhabhishek22957/portfolio.git
```

Navigate into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Create a `.env.local` file

```env
RESEND_API_KEY=your_resend_api_key
```

Run the development server

```bash
npm run dev
```

Open your browser

```text
http://localhost:3000
```
